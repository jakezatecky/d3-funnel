d3-funnel

A JavaScript library for rendering funnel charts using the D3.js framework.

d3-funnel is an extensible, open-source JavaScript library for rendering funnel charts using the D3.js library.

d3-funnel is focused on providing practical and visually appealing funnels through a variety of customization options. Check out the example below to get a showcasing of the several possible options.
